/** * 醫護實習生一站式服務平台 - 後端接收程式 * 功能：接收來自網頁的 POST 請求，將課程報名與實習登記資料寫入 Google Sheets */ const SHEET_NAME = "報名資料"; // 試算表分頁名稱 function doPost(e) { const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName(SHEET_NAME); // 如果找不到分頁，則建立一個 if (!sheet) { sheet = ss.insertSheet(SHEET_NAME); setupHeaders(sheet); } try { const data = JSON.parse(e.postData.contents); const timestamp = new Date(); // 根據資料類型 (type) 整理寫入的列 // 欄位順序：提交時間, 資料類型, 姓名, 學號/員編, 單位/學校, Email, 課程名稱, 性別, 身分證/護照, 實習科別, 開始日期, 結束日期, 手機, 緊急聯絡人 let row = [ timestamp, data.type, // 課程報名 或 實習登記 data.name || "", data.idNumber || "", // 學號或員編 data.unit || "", // 單位或學校 data.email || "", data.courseName || "N/A", // 僅課程報名有值 data.gender || "", // 僅實習登記有值 data.identityId || "", // 僅實習登記有值 data.department || "", // 僅實習登記有值 data.startDate || "", // 僅實習登記有值 data.endDate || "", // 僅實習登記有值 data.phone || "", // 僅實習登記有值 data.emergency || "" // 僅實習登記有值 ]; sheet.appendRow(row); return ContentService.createTextOutput(JSON.stringify({ "result": "success" })) .setMimeType(ContentService.MimeType.JSON); } catch (error) { return ContentService.createTextOutput(JSON.stringify({ "result": "error", "error": error.toString() })) .setMimeType(ContentService.MimeType.JSON); } } // 初始化表頭的功能 function setupHeaders(sheet) { const headers = [ "提交時間", "資料類型", "姓名", "學號/員編", "單位/學校/系所", "Email", "課程名稱", "性別", "身分證/護照", "實習科別", "開始日期", "結束日期", "手機號碼", "緊急聯絡人與電話" ]; sheet.getRange(1, 1, 1, headers.length).setValues([headers]); sheet.getRange(1, 1, 1, headers.length).setBackground("#134E4A").setFontColor("#FFFFFF").setFontWeight("bold"); sheet.setFrozenRows(1); } // 執行此函式可手動建立表頭 function runSetup() { const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) sheet = ss.insertSheet(SHEET_NAME); setupHeaders(sheet); }
