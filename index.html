/** * 醫護實習生一站式服務平台 - 後端接收程式 * 建議：請先執行一次 runSetup 函式來建立試算表表頭 */ const SHEET_NAME = "報名資料"; // 請確保試算表內的分頁名稱完全一致 // --- 1. 處理網頁傳來的資料 (POST) --- function doPost(e) { const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) { sheet = ss.insertSheet(SHEET_NAME); setupHeaders(sheet); } try { const data = JSON.parse(e.postData.contents); const timestamp = new Date(); // 欄位順序對應表 const row = [ timestamp, data.type || "未分類", data.name || "", data.idNumber || "", data.unit || "", data.email || "", data.courseName || "N/A", data.gender || "", data.identityId || "", data.department || "", data.startDate || "", data.endDate || "", data.phone || "", data.emergency || "" ]; sheet.appendRow(row); return ContentService.createTextOutput(JSON.stringify({ "result": "success" })) .setMimeType(ContentService.MimeType.JSON); } catch (error) { return ContentService.createTextOutput(JSON.stringify({ "result": "error", "message": error.toString() })) .setMimeType(ContentService.MimeType.JSON); } } // --- 2. 讓網址可以直接打開看狀態 (GET) --- // 增加這個函式後，你點開網址才不會看到錯誤，而是會看到「服務運作中」 function doGet() { return ContentService.createTextOutput("服務運作中！請透過網頁表單提交資料。") .setMimeType(ContentService.MimeType.TEXT); } // --- 3. 初始化表頭設定 --- function setupHeaders(sheet) { const headers = [ "提交時間", "資料類型", "姓名", "學號/員編", "單位/學校/系所", "Email", "課程名稱", "性別", "身分證/護照", "實習科別", "開始日期", "結束日期", "手機號碼", "緊急聯絡人與電話" ]; sheet.getRange(1, 1, 1, headers.length).setValues([headers]); sheet.getRange(1, 1, 1, headers.length) .setBackground("#134E4A") .setFontColor("#FFFFFF") .setFontWeight("bold"); sheet.setFrozenRows(1); } // --- 4. 手機手動執行此函式來初始化 --- function runSetup() { const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) { sheet = ss.insertSheet(SHEET_NAME); } setupHeaders(sheet); }
